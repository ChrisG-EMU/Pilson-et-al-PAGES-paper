# Pilson-et-al-PAGES-paper
[![License: CC0-1.0](https://img.shields.io/badge/License-CC0%201.0-lightgrey.svg)](https://creativecommons.org/publicdomain/zero/1.0/)

This site supports the manuscript by Olivia Pilson et al. entitled "Developing Targeted Genetics Educational Resources to Improve Healthcare Providers Communication on Genetic Testing Options for Autism".

## Repository structure

- **R code/**: scripts used for data cleaning, analysis, and figure generation.
- **statistics/**: statistical model outputs, regression tables, post-hoc comparisons.
- **supplementary information/**: supplementary figures, tables, and other supporting materials.

## Data availability

Data associated with this manuscript may be shared publicly if/when added to this repository. At present, this repository contains analysis and figure-generation code only.

## How to use

- Clone the repository
- Navigate to `R code/` to review the analysis script
- See `statistics/` for regression and statistical outputs, if/when added
- See `supplementary information/` for additional tables and figures, if/when added
